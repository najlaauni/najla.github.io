The third lab exercise had the requirement that I code a complete program in C++ to tell the COVID-19 zone status by the number of active cases. It was a very helpful exercise for me to see that programming usefulness goes beyond mere computer scientists and is the case especially with the public health decisions by government authorities.

The lab was very intensive but at the end of it, I was very much more skilled to use logical comparisons and conditions to do accurate zone classifications of calculated values. 'Boring' but needed steps of translating the problem statement and expected output into a C++ program worked as a drill for my ability to analyse the requirements and implement them in the code correctly.

Even though the time for the lab was quite limited, it made me solve problems more efficiently and develop my debugging skills. The primary reason behind changing my debugging technique was to ensure that the program was error free and gave the correct output. However, the lab task made me more confident about writing C++ structured programs and it also revealed that accuracy and clarity in programming are the main tools for solving real-life problems.
